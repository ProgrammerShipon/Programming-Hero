    Flex-box vs grid-box difference?

    Flexbox and Grid are both CSS layout modules that allow developers to create flexible and responsive layouts.

    • Flexbox is used for one-dimensional layout, where elements are arranged in a single row or column. It is best suited for small-scale layouts, such as navigation menus or form elements.
    • Grid, on the other hand, is used for two-dimensional layout, where elements are arranged in rows and columns. It is best suited for larger-scale layouts, such as entire web pages or sections of a web page. Grid allows for more complex layouts and positioning of elements.

    Both flexbox and grid have their own specific use cases, and developers can use them together for more advanced layouts.

    What is the use of media query?

    Media queries are a feature of CSS that allows developers to apply different styles to a document based on the properties of the device displaying the document, such as screen size, screen resolution, and device type.

    They allow web designers to create responsive web designs that adapt to different screen sizes and resolutions, providing an optimal viewing experience for the user.

    What is the CSS box model?

    In web development, the CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages.

    The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
